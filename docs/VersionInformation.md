# 版本資訊

## 2.3.1 — 2026-07-22

- 遊戲介紹改為遊戲規則、得分機制、遊玩方式、排行榜四個區塊。
- README 使用相同順序詳細說明 10×10 棋盤、滑動、方塊放置、消除得分、操作與排行榜。
- README 新增由 Solar2D 實際遊戲場景輸出的完整遊戲畫面截圖。
- 保留 GitHub Releases 累計下載次數徽章。
- 新增遊戲介紹內容順序、關鍵規則、README 徽章與截圖一致性測試。

## 2.3.0 — 2026-07-22

- 本機與全球排行榜統一改為每頁 10 個名次。
- 排行榜底部新增上一頁、目前頁碼與下一頁操作。
- 新增可共用的純分頁模組，資料服務、應用流程與 Solar2D 畫面保持分離。
- 新增空榜、首頁、中間頁、末頁、越界頁碼及本機／全球換頁整合測試。

## 2.2.1 — 2026-07-22

- Game Over 的重新開始與回到主畫面改為垂直排列，並限制在兩個預覽框之間。
- 0 分不再寫入本機排行榜。
- 個人排行榜改名為本機排行榜，並彙整裝置上所有帳號的遊玩紀錄。
- 新增跨帳號彙整、0 分排除、刪除定位、版面邊界與個資不落地測試。

## 2.2.0 — 2026-07-22

- 所有選項改為高亮度粗體文字。
- 遊戲中新增回到主畫面按鈕，返回時保存當前分數。
- Game Over 新增重新開始與回到主畫面兩個選項。
- 新增忘記密碼信、登入後修改密碼與修改暱稱。
- 新增 Refresh Token 工作階段恢復；不保存明文密碼或 ID Token。
- 新增 Android 背景恢復、貼圖重建及失敗時回主畫面的保護流程。
- 新增資安、個資隱私、生命週期、密碼與分數防重複測試。

## 2.1.0 — 2026-07-21

- 全球排行榜改為每個帳號只保留並顯示最高分。
- 新增玩家暱稱設定，個人與全球排行榜改顯示暱稱。
- 修正旋轉預覽後移動時，實際方塊仍以未旋轉形狀落下的問題。
- 新增手機四方向滑動手勢，對應鍵盤 W、A、S、D。
- 擴充最高分去重、暱稱同步、旋轉放置與滑動邊界測試。

## 2.0.0 — 2026-07-21

- Game Over 後改為回到封面。

- 新增 Firebase Email/Password 登入與 Firestore 全球排行榜。
- 新增依帳號隔離、可逐筆刪除的本機排行榜。
