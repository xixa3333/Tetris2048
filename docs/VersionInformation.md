# 版本資訊

## 2.3.7 — 2026-07-23

- 封面於退出遊戲上方新增「設定」。
- 設定頁新增背景音樂與消除音效 0～100% 滑桿，並即時保存到本機。
- 背景音樂只在遊戲進行中播放；返回封面、Game Over 或 App 暫停時會停止。
- 新增最多 64 字元的關卡種子；相同種子每次重開都能重現相同方塊與落點順序，留空維持一般隨機。
- 保留全域 letterbox 自動縮放；設定頁與封面共用相同 500×850 版面，修正元件位移。
- 封面右下角顯示由版本資訊模組提供的目前版本號。
- 啟動時檢查 GitHub Releases；發現較新版本才提示，並導向固定的 `/releases/latest` 連結。
- 更新檢查失敗或離線時會靜默略過，不影響封面與離線遊玩。
- 合法落點會完整搜尋目前旋轉方向的所有座標，並忽略形狀外圍純空白邊界；只有目前方向完全無法放置時才結束遊戲，不會自動旋轉。
- 將設定、音訊及亂數拆成獨立服務，維持低耦合與可測試性。
- 新增音量、靜音、種子重現、合法落點、旋轉方向、生命週期、響應式介面與異常設定資料測試。

## 2.3.6 — 2026-07-23

- 修正啟用 Firebase 電子郵件列舉防護後，帳號 ID 修改必定失敗的問題。
- 新註冊 ID 改為建立後永久不可修改，虛擬信箱由程式自動產生且不對玩家顯示。
- 舊電子郵件帳號登入後顯示一次性 ID 轉換，不影響一般 ID 帳號。
- 轉換前重新驗證目前密碼，並確認新 ID 未被使用。
- 成功轉移暱稱與全球最高分後才刪除舊排行榜紀錄；中途失敗會清理新資料並恢復舊登入。
- 不使用 Cloud Functions、不需 Blaze 方案，也不把真實電子郵件放進公開 Firestore 對照表。
- 新增永久 ID、重複 ID、成功轉移與失敗回復測試。

## 2.3.5 — 2026-07-23

- 新增固定 3×3 矩陣的藍色 L 方塊，並完整支援四方向旋轉、碰撞檢查及隨機放置。
- 封面新增「APP 資訊」，提供遊戲 GitHub、問題回報區、逐版本修正摘要與作者 GitHub。
- 全球排行榜固定顯示登入玩家自己的完整全球名次，並加亮名次提示及自己的排行榜列，不必逐頁尋找。
- 新帳號改用具唯一性的 3～20 字元 ID，不再要求電子郵件；既有信箱帳號可繼續登入並改成一般 ID。
- 帳號 ID 可在登入後修改，唯一性由 Firebase Authentication 原子檢查。
- Firestore 停止保存帳號 ID 與電子郵件，只保留 UID、暱稱及分數等排行榜必要資料。
- 新增帳號格式、重複 ID、舊帳號遷移、全球名次、資料最小化及 L 方塊旋轉邊界測試。

## 2.3.4 — 2026-07-23

- 修正不同顏色相連元件交錯滑動時，後處理顏色可能覆蓋先處理顏色的規則錯誤。
- 滑動改為共用即時占用表並逐格推進，任何時刻都不允許兩個元件占用同一格。
- 動畫移動資料新增唯一目的地驗證，避免不同顏色動畫落在相同格子。
- 新增固定覆蓋案例、250 組高密度隨機棋盤及 1,000 回合自動試玩測試。

## 2.3.3 — 2026-07-23

- 將單回合拆分為移動、第一次消除、方塊放置、第二次消除四個連續動畫階段。
- 整段回合動畫期間鎖定鍵盤、按鈕與滑動輸入，避免重複操作造成狀態不同步。
- 動畫交接先渲染下一個棋盤狀態再移除前一層，修正方塊閃爍、消失後重現的視覺問題。
- 加入回合識別與過期計時器防護，避免重新開始後舊回呼修改新棋盤。
- 保留交易式落點檢查，並新增階段順序、輸入鎖、動畫邊界及過期回呼測試。

## 2.3.2 — 2026-07-23

- 方塊落地改為交易式放置：寫入棋盤前會再次逐格確認位置仍為空白。
- 若預定落點已被占用，會改試其他合法落點；全部失效時安全結束遊戲，不覆蓋既有方塊。
- 新增落點競態、部分寫入、障礙位置矩陣與控制器整合測試。
